## 1. Cara Membatasi Bandwidth Wifi Speedy Tanpa Software
 
Cara ini bisa kita lakukan dengan menggunakan fitur bawaan dari Windows, yaitu Local Group Policy Editor. Fitur ini memungkinkan kita untuk mengatur berbagai pengaturan di komputer kita, termasuk pengaturan bandwidth. Berikut adalah langkah-langkahnya:
 
1. Buka kotak dialog Run dengan menekan tombol Windows + R pada keyboard, lalu ketik `gpedit.msc` dan tekan Enter.
2. Pada kotak dialog Local Group Policy Editor, di panel kiri pilih folder Computer Configuration > Administrative Templates > Network > QoS Packet Scheduler.
3. Pada panel kanan, klik dua kali pada opsi Limit reservable bandwidth, lalu pilih Enable dan atur berapa persen bandwidth yang ingin Anda batasi dengan mengisi angka pada kolom Bandwidth limit (%). Angka ini menunjukkan berapa persen bandwidth yang akan dibagikan ke perangkat lain, sedangkan sisa persennya akan digunakan oleh komputer Anda.
4. Klik Apply dan OK untuk menyimpan perubahan.
5. Restart komputer Anda untuk menerapkan pengaturan.

## 2. Cara Membatasi Bandwidth Wifi Speedy dengan Software P2POver
 
Cara ini bisa kita lakukan dengan menggunakan software P2POver, yaitu sebuah aplikasi yang berfungsi untuk mengatur bandwidth wifi Speedy seperti sebuah router. Aplikasi ini bisa mendeteksi semua perangkat yang terhubung ke wifi Speedy dan mengatur kecepatan internetnya sesuai dengan keinginan kita. Berikut adalah langkah-langkahnya:

1. Download dan install software P2POver di komputer Anda.
2. Buka aplikasi P2POver dan klik tombol Scan untuk memindai semua perangkat yang terhubung ke wifi Speedy.
3. Pilih perangkat yang ingin Anda batasi bandwidthnya dengan mengklik nama atau alamat IP-nya.
4. Atur berapa besar kecepatan upload dan download yang ingin Anda berikan kepada perangkat tersebut dengan menggeser slider pada kolom Limit Upload/Download.
5. Klik Apply untuk menyimpan pengaturan.
